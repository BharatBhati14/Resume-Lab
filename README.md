# 🚀 ResumeLab

ResumeLab is an AI-powered full-stack resume builder that helps users create professional, ATS-optimized resumes in minutes using smart AI assistance.

It includes resume building, AI-powered writing improvements, authentication, and resume management.

---

## ✨ Features

### 🧠 AI Features
- Generate professional resumes using AI
- Improve bullet points instantly
- Create ATS-optimized summaries
- Suggest skills based on job roles
- Tailor resumes for job descriptions

### 📄 Resume Builder
- Drag & edit resume sections
- Live resume preview
- Multiple templates
- PDF export

### 🔐 Authentication
- User signup/login
- JWT-based authentication
- Protected routes

### 📊 Resume Management
- Create many resumes
- Edit & delete resumes
- Save resume drafts
